Arduino IDE - среда разработки, предназначенная для программирования микроконтроллеров на платах Arduino

Arduino IDE 2.0 можно скачать с оф сайта  https://www.arduino.cc/

*************
Подключение платы и загрузка кода в плату
**************
1) скачать и установить IDE
2) подключить к USB плату Arduino через кабель
	* если IDE устанавливался с инсталлятора, то драйвер для Arduino уже установлен иначе его надо будет найти на оф сайте (у меня поставился автоматом)
3) Плата Ардуино будет отображаться в диспетчере устройств: Порты (COM и LPT)
	* у меня оно называлось так "Устройство с последовательным интерфейсом USB (COM4)"
4) В IDE нужно написать некий код (либо взять готовый пример из File -> Examples)
5) Далее нужно указать Плату и номер порта для которых будет загружаться код
	* Возможно потребуется сначала скачать Arduino пакет из Boards Manager (я ставил Arduino AVR Boards v1.8.4)
	* Зайти в Tools -> Board и выбрать плату, а затем в Tools -> Port и выбрать порт (у меня это COM4)
	   (либо под Help восопользоваться дропдауном, который уже сам определил тип подключенной платы + порт)
6) Нажать Upload для загрузки программы (после загрузки, программа сразу же начнет исполняться на плате)
